# prusaslicer
A docker image to compile PrusaSlicer from source.

## Example Usage:
```
git clone git@github.com:tagadvance/prusaslicer.git
cd prusaslicer
echo "alias prusaslicer=$(pwd)/prusaslicer.sh" >> ~/.bashrc
prusaslicer
```
